## 安装插件

- vue-router 管理路由菜单
- axios 进行数据请求
- elementui 进行页面布局
- md5 对用户的密码和用户id加校验(加密)，在数据库只保持这个校验值，每次登陆都对这个校验值和数据库的校验值比对，相同即可通过，确保拦截
- wangeditor 富文本编辑插件
